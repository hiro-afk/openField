# openField
オープンフィールドの解析用プログラムです。

### 手順①:RとRstudioのインストール
まず初めにRをダウンロードします。Rは統計解析に特化したプログラミング言語です。\
Rを快適に使用するためにはRとRstudioのインストールが必須です。\
Rは以下のURLからインストールしてください。\
https://cran.rstudio.com/bin/windows/base/ \
URLを開いたら　Download R-(バージョン名) for Windows (Mac)\
